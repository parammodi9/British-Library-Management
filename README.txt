************************************************************
SOFTWARE REQUIREMENTS:
NETBEANS 8.0
MYSQL 5.5
JDK KIT
************************************************************
Steps to use LibSol:
************************************************************
IN MYSQL:
1) Open MySQL create Database britishlibrary.
2) Create 3 tables in it.
3) Create table login with fields
	-username (varchar)
	-password (varchar)
4) Create table addbooks with fields
	-book_name
	-book_id
	-author
	-edition
	-publisher
5) Create table issuebooks with field
	-student_name
	-student_id
	-book_name
	-book_id
	-author
	-edition
	-publisher
************************************************************

IN NETBEANS:
1) Create a new project named britishlibrary.
2) go to the destination folder of the project and copy the src file from the libsol project folder to the src folder.
